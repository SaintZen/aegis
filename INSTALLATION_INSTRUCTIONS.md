# Installation Instructions - v2.6

## ⚠️ **IMPORTANT: Uninstall First**

If the app isn't updating properly, **uninstall the old version first**, then install the new one.

---

## 📱 **Installation Steps**

### **Option 1: Uninstall Then Install (RECOMMENDED)**

1. **On your phone:**
   - Go to Settings > Apps > Anxiety Anchor
   - Tap "Uninstall"
   - Confirm uninstall

2. **Transfer the new APK:**
   - Copy `AnxietyAnchor-v2.6-FORCE-UPDATE.apk` to your phone
   - Location: `dist\AnxietyAnchor-v2.6-FORCE-UPDATE.apk`

3. **Install:**
   - On your phone, open the APK file
   - Tap "Install"
   - Allow installation from unknown sources if prompted

### **Option 2: Direct Update (If Option 1 doesn't work)**

1. **Transfer APK to phone**
2. **Open APK file**
3. **Tap "Install"**
4. **If it says "App not installed" or "Package invalid":**
   - Go to Settings > Apps > Anxiety Anchor
   - Uninstall the app
   - Then install the new APK

## ✅ **What's New in v2.6**

- ✅ **Voice Support for Affirmations** - 🔊 Play button on each affirmation
- ✅ **Voice Support for Rescue Chat** - Toggle switch for automatic speech
- ✅ **Fixed Progress Charts** - Charts should now load properly
- ✅ **Color-Coded Affirmations** - Each sentence has a different color (20sp, bold)
- ✅ **Fixed Breathing Circle Colors** - All phases now show distinct colors

---

## 🔍 **How to Verify Features**

### **1. Affirmations Voice:**
- Open "Affirmations Library"
- Look for "🔊 Play" button at bottom of each card
- Tap it to hear the affirmation spoken

### **2. Affirmations Color Coding:**
- Open "Affirmations Library"
- Each sentence should be a different color (Blue, Green, Orange, Purple, etc.)
- Text should be larger (20sp) and bold

### **3. Progress Charts:**
- Open "Progress & Insights"
- Charts should display (may show "No data available" if you haven't created logs yet)
- Create some anxiety logs in "Anxiety Lab" first to see data

### **4. Breathing Circle Colors:**
- Open "Panic Response" or "Anchor Me Now"
- Start breathing exercise
- Circle should change colors:
  - Blue for INHALE
  - Green for HOLD IN
  - Purple for EXHALE
  - Warm Amber for HOLD OUT

---

## 🐛 **If Features Still Don't Work**

1. **Uninstall the app completely**
2. **Restart your phone** (optional but recommended)
3. **Install the new APK**
4. **Check app version:**
   - Should show "v2.6" in app name
   - Settings > Apps > Anxiety Anchor should show version 2.6
